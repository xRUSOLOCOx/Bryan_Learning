Mi titulo

El sistema de juego de Baseball Heroes es muy sencillo: solamente tendremos que presionar sobre la pantalla de nuestro dispositivo para indicar nuestra intención de lanzar la pelota de béisbol. Como es lógico, cuanto más lejos llegue la pelota, mayor puntuación acumularemos. Sin embargo, a diferencia de otras obras, inmediatamente después de lanzarla, seremos nosotros mismos los que deberemos de recibirla y batearla (lográndolo haciendo de nuevo ‘tap’ sobre la pantalla en el momento adecuado).
